# IGC - Instagram Crawler

Crawler automatizado para extrair dados de perfis do Instagram usando automação de browser.

## 🛠️ Stack

- **Backend**: Python + FastAPI + Playwright
- **Frontend**: Vue.js + Vite
- **Objetivo**: Aprendizado de web scraping e automação

## 📦 Funcionalidades

- Login com conta real do Instagram
- Extração de dados do perfil (nome, bio, foto, stats)
- Lista completa de seguidores (nome, username, foto, link)
- Lista completa de seguindo
- Interface web para visualização

## ⚠️ Aviso

Projeto educacional. Use por sua conta e risco. Automação do Instagram pode violar os termos de serviço.
